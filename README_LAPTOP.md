# Laptop Nutzung

## Projekt starten
1. Ordner auf dem Laptop in VS Code öffnen
2. Terminal im Projekt öffnen
3. Abhängigkeiten installieren:
   npm install
4. Dev Server starten:
   npm run dev

## Wichtige Hinweise
- .git wurde mitkopiert, das Repo bleibt also erhalten
- .env Dateien sollten mitkopiert worden sein
- node_modules wurde absichtlich nicht kopiert
- Falls npm Fehler macht:
  - package-lock.json behalten
  - einfach nochmal npm install ausführen

## Prüfen
- Git Status:
  git status
- Node Version:
  node -v

## Empfehlung
Für sauberes Arbeiten den Ordner vom externen Laufwerk lieber auf die SSD des Laptops kopieren und dort weiterarbeiten.
